## SYNOPSIS

**SphereSSL** is a modern, user-friendly SSL certificate manager with a web dashboard built with ASP.NET. It makes SSL certificate creation and renewal effortless for everyone, whether you're a hobbyist or a professional.

*"One cert manager to rule them all, one CA to find them, one browser to bring them all, and in encryption bind them."*

## FEATURES

### Certificate Management
- **Quick Create Flow**: Simple, intuitive certificate creation process
- **Auto & Manual DNS**: Automated DNS record creation or manual entry for any provider
- **Renewal Scheduling**: Toggle auto-renew on/off with renewal notifications
- **Import/Export**: Convert, download, or upload certs in your preferred formats

### Supported DNS Providers
- AWS Route53
- Cloudflare
- Cloudns.net
- DigitalOcean
- DNSMadeEasy
- DreamHost
- DuckDNS
- Gandi
- GoDaddy
- Hetzner
- Linode
- Namecheap
- Porkbun
- Vultr
- ...and more coming soon!

### Enterprise Features
- **Unlimited Domains**: No arbitrary limits
- **Multi-User Support**: Share certs/orders for collaborative management
- **Provider Integrations**: Tons of DNS provider integrations

## DATA STORAGE

- **Data**: Application database and settings
- **Certs**: Generated SSL certificates
- **Logs**: Application logs
