# Wealthfolio

Wealthfolio is a local-first investment tracking app that helps you manage your portfolio, analyze performance, and set financial goals. It is cross-platform and stores all data locally, ensuring privacy and security.

## Features

- **Portfolio Tracking**: Track investments across multiple accounts and asset types.
- **Performance Analytics**: Detailed performance metrics and historical analysis.
- **Goal Planning**: Set and track financial goals with allocation management.
- **Local Data**: All data stored locally with no cloud dependencies.
- **Multi-Currency Support**: Manage investments in multiple currencies.
- **Extensible Addon System**: Add custom functionality with Wealthfolio's addon SDK.
